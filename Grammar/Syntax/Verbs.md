# Verbs
---
## Mood
Each noun can be turned into a verb using the mood specifier as verb marker. The language have three mood specifier: 
- **ni**: indicative 
- **ka**: interrogative 
- **go**: imperative 

Using "kome" (consumable) you can form the verb "to consume" also used as "to eat": 
> "Mi **ni kome**" -> "I eat"
> "Tu **ka kome**?" -> "do you eat ?"
> "**go kome**" -> "eat" as a order

Imperative mood will always refer to second person.

---
## Valency
All verbs expect subject as argument (there is no impersonal verbs). Verbs are divided in 3 type: 
### Unitary
 Expect only the subject:
> "**Mi** ni sapa" -> "**I** sleep"
> "**le** ni guoka" -> "**She/He** work"
### Binary patient
 Expect subject and direct object (patient), the thing that is being affected by the action. You have to use the direct object marker "ke" to indicate the object:
> "**Mi** ni kome **ke patao**" -> "**I** eat **duck**"
### Binary destination
Expect subject and indirect object (destination), the goal. You have to use the indirect object marker "to" to indicate the object:
> "**Mi** ni gona **to kasaca**" -> "**I** go **to house**"
### Ternary
Expect subject and both direct and indirect object. You may use both object markers to indicate each object: 
> "**Mi** ni dona **ke kanao** ***to tu***" -> "**I** give **the dog** ***to you***"

Some languages use a particle to indicate the indirect object like the "to" marker in English but don't indicate direct object.

---
## Auxiliarly verbs 
You can link verbs together to modify mood: 
> "mi **ni guane ni kome**" -> "I **want to eat**"
> "tu **ni nese ni gona**" -> "You **need to go**"

### Voice modifier
There are two particles used to modify the voice: 
- **ni kea**: Passive indirect object 
- **ni toa**: Passive direct object 

Examples: 
> "patao **ni kea** kome bi mi" -> "The duck was eaten by me"
> "kasaca **ni toa** gone bi mi" -> this doesn't have a literal translation in English but it should be something like "The house was go by me"
