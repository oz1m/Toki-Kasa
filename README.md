# Toki Kasa — A Constructed Language

**Toki Kasa** is an open and collaborative conlang (constructed language) designed with one central goal:  
**Keep everything simple, consistent, and logical.**

- No irregularities  
- No hidden complexity  

---

## Core Principles

PC is built on a few foundational ideas:

- **Phonetic simplicity:** All words follow predictable CV/CVV syllable structures.
- **Logical grammar:** Rules should be minimal, universal, and easy to apply.
- **Consistency:** Every feature must follow from a clear internal logic.
- **Community-driven evolution:** Proposals and discussion shape the language.
